# Changelog

All notable changes to MyVMR will be documented here.

## [1.0.0] - 2026-05-13

### Added
- Initial release
- Auto-detect FSLTL, AIG OCI, IVAO MTL and JustFlight FSTraffic paths for FS2020 and FS2024
- FSLTL VMR merge to ensure unmatched traffic falls back to FSLTL generic models
- Model Provider Preference to prioritise models from a chosen library
- Custom Rules page to add, edit and remove VMR rules
- Log viewer modal showing generation output
- Collapsible sidebar
